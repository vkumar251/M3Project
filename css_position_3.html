Positions

Relative Positions

Relative positioning allows you to position an element relative to its normal position in the document flow. position:relative; is the syntax for applying relative positioning to an element.

Static Positions

Static positioning is used when you want an element to stay in the same position relative to its normal position in the document flow. position: static; is the default position for all elements.

Absolute Positions

Absolute positioning is used when you want an element to be positioned relative to its closest positioned ancestor (not the body). position: absolute; is the syntax for applying absolute positioning to an element.

Fixed Positions

Fixed positioning is used when you want an element to stay in the same position relative to the viewport, even when scrolling. position: fixed; is the syntax for applying fixed positioning to an element.

Sticky Positions

Sticky positioning is used when you want an element to be positioned relative to the viewport, but remain fixed as the user scrolls. position: sticky; is the syntax for applying sticky positioning to an element.
